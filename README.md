 # FreshFruits - Online Fruit Shop

## Project Description
FreshFruits is a modern web application built with Next.js and React for buying and viewing fresh fruits online. The project demonstrates authentication, product listing, and interactive UI features.

Route	Description
i. Home page with Navbar

vi. aboutSection	About page
v. contactPage	Contact page

## Implemented Features
User authentication (mock login)
Navbar that dynamically updates based on login state
Responsive design for mobile and desktop
Product listing with images and details
Smooth scroll effect with hiding/showing navbar
Toast notifications for successful actions

## Feature Explanation
User Authentication: Users can log in; the navbar button changes from LOGIN to ALL CARDS.
Navbar Scroll Effect: Hides on scroll down and shows on scroll up.
Responsive Design: Works well on mobile, tablet, and desktop devices.
Product Cards: Each product displays an image, name, price, stock, and description.
Route Management: Clean navigation with Next.js Link for routing.
